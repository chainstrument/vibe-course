# Claude Instructions - Vibe Course

## Vision du projet

Le vibecoding n'a pas tué le métier de développeur — il l'a **ouvert à tout le monde**. N'importe qui peut désormais s'asseoir et construire l'application qu'il veut, sans savoir coder. Cette plateforme enseigne précisément cela : pas à coder, mais à **builder**.

## Ce qu'on enseigne

Deux axes complémentaires :

1. **Théorie** — comprendre les concepts clés sans les coder (frontend/backend/BDD/API), penser une application avant de la construire, structurer ses prompts
2. **Pratique** — exercices concrets avec énoncés et corrigés statiques. L'apprenant utilise l'IA de son choix (Claude.ai, ChatGPT, Cursor) en dehors de la plateforme

## Public cible

Débutants complets. Aucune connaissance en code requise.

## Contraintes

- **Pas de clé API** : la plateforme n'intègre pas d'IA. Elle enseigne à utiliser les outils IA existants (Claude.ai, ChatGPT, Cursor...) comme outils externes
- **Pas monétisée** pour l'instant — stack gratuite uniquement
- **Contenu statique** : les leçons et corrigés sont des fichiers MDX, pas de backend dynamique en V1

## Stack

- **Frontend** : Next.js + MDX (leçons en markdown) + Tailwind CSS
- **Hébergement** : Vercel (gratuit)
- **Pas de base de données en V1** (pas de suivi de progression utilisateur)
